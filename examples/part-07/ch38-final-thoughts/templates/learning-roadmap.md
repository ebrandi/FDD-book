# Personal Learning Roadmap

Use this template to plan each three-to-six-month learning
cycle. Copy the template, fill it in, and keep the filled copy
in your `freebsd-learning` repository so you can trace your
trajectory over time.

A roadmap is not a contract with yourself. It is a working
plan that you will revise as circumstances change. The value is
in the planning and the revising, not in the strict adherence.
