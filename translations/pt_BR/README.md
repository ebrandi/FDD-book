# Brazilian Portuguese Edition (pt_BR)

This directory contains the Brazilian Portuguese translation of the book
*FreeBSD Device Drivers: From First Steps to Kernel Mastery*. The
disclaimer below is provided first in English and then in Brazilian
Portuguese so that readers coming from either language can evaluate the
status of this translation before reading the book.

---

## English

> **Disclaimer: AI-Assisted Translation**
>
> This Brazilian Portuguese edition of *FreeBSD Device Drivers: From
> First Steps to Kernel Mastery* was produced using AI-assisted
> translation. The English manuscript under `content/` was translated
> automatically by a local pipeline built on Ollama, the
> `qwen3.6:35b-a3b-bf16` model, and Python scripts that drive the
> translation workflow.
>
> **This translated content has not yet been fully reviewed by the
> author.** It may still contain translation errors, terminology
> inconsistencies, awkward phrasings, or passages that do not sound
> natural in Brazilian Portuguese. Technical terms, code snippets,
> command names, kernel API references, and FreeBSD-specific vocabulary
> may also deviate in places from the conventions the author prefers.
>
> If any part of the translation is unclear, ambiguous, or appears to be
> incorrect, please treat the **original English version under
> `content/` as the authoritative source of truth**. Feedback,
> corrections, and pull requests that improve the translation are warmly
> welcomed and will be incorporated as the review progresses.
